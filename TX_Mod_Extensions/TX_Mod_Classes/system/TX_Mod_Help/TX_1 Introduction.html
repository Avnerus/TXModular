1. Background

This system was designed and written between 2002 and 2013. It started out as a few ideas for a MIDI-controlled synth and quickly expanded to become a modular system.


It includes various modules, such as: sample-players, loop players, envelope generators, multi-waveform synths, filters, noise generators, LFOs, delays, compressor/expanders, flangers, pitch-shifters, reverbs, vocoders, distorters, ring modulators file recorders, file players, and step sequencers. Click for a full list: Intro to All Modules


External controllers can be connected the system - such as a MIDI keyboard, slider, joystick, iPad, Wii Remote, graphics tablet, ribbon controller, or foot pedal. OSC (Open Sound Control) messages can be sent to and from the system across a local network or through the internet. Visual compositions created using Quartz Composer can be displayed and controlled directly from within the system.


The TX Modular System is written in the SuperCollider programming language (http://supercollider.sourceforge.net). However, you don't need to know how to program in SuperCollider to use this system. If, on the other hand, you can program in SuperCollider, there are various modules which allow you to add your own code, e.g. for audio generation or animation.


I hope there is enough information for you in these help files to begin experimenting and creating.
