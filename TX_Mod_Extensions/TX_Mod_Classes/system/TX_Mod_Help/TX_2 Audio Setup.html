The TX Modular System - Audio Setup Window


When the TX Modular System is started, the Audio Setup window is displayed first. This is where you configure your audio settings, before starting the audio engine.

The controls are as follows:


Start Audio Engine button - this will start the engine and display the main TX Modular System window. Only press this once you have set the options for your audio device - see below.


Quit button - This quits the TX Modular System completely.


Audio Device For Input and Output - This sets the audio device that will be used for audio input & output to & from the the TX Modular System.

- If you set this to Use Default, it will use the default device for your computer. To check or change the default audio device, run Audio MIDI Setup which can be found in the folder: Applications/ Utilities. Select on the Audio Devices window and see which device is marked with symbols for sound input and output.

Note for Mac OSX: You won't be able to start the audio engine if you select built-in microphone, built-in input, or built-in output as the Audio Device because they aren't valid as inputs and outputs (they are just one or the other). To use your built-in microphone or built-in input for input and your built-in output for output, you need to first create a virtual "aggregate" device using Audio MIDI Setup. (See notes below).


Audio Device Sample Rate - If you set this to Use Default, it will use the default settings for the device. If you change it to another value, make sure it is valid for that device.


Audio Device Buffer Size - If you set this to Use Default, it will use the default settings for the device. If you change it to another value, make sure it is valid for that device.

Note for Mac OSX: Creating an Aggregate Device in Audio MIDI Setup

To use your built-in microphone and/or built-in input for input and your built-in output for output, you need to create a virtual "aggregate" device (a virtual device that wraps two or more other devices).


If you create an aggregate of your built-in microphone, built-in input and built-in output, you can use this single aggregate device as the Audio Device for the TX Modular System. You need to set up this aggregate device only once.


To do so, you have to open the Audio MIDI Setup application (in /Applications/Utilities) â€“ you should do this from an user account with administrator privileges.

On the Audio Devices window click on the plus [+] button in the bottom left corner to add a new aggregate device.

Double click on its name in the list (by default "Aggregate Device") and change it to a more specific name, for example "Built-in Audio". Then check the Use column for the devices you want to combine, that is your microphone and/or line input and the built-in output.

Right click on the new aggregate device and select both "use this device for sound input" and "use this device for sound output".


Then close Audio MIDI Setup.
