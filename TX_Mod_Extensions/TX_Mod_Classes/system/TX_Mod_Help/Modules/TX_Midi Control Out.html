Midi Control Out


Module type:  Action


Introduction:

This module sends out MIDI controller messages. 

There are 128 possible MIDI controller numbers, which can be sent on one of 16 MIDI channels.

The value of the control signal can be between 0-127.

Make sure to switch the  Active button on so MIDI messages are sent out.

In order to interact with other software, it is sometimes useful (for example when using "midi learn"  in some software) to be able to temporarily solo one Midi Control Out module, so midi data for only one controller number will be sent out. This is called Solo Mode and can be controlled using buttons (see below).


Controls:

Port - sets the MIDI port messages are sent from. 

Data rate - the maximum rate of sending MIDI messages. The default is 20, but make this 40 or 80 if you are using quickly changing modulation (such as an LFO with a frequency higher than 5 hz).

Channel - sets the MIDI channel (1-16) messages are sent on.

Control no - sets the MIDI controller number (0-127) messages are sent as.

Control val - sets the controller value (0-127)  to be sent. This value can also be modulated.

Active - if switched on, MIDI messages will be sent out. 


Global Controls:

Turn off Solo Mode for all Midi Control Out modules - button

Turn on Solo Mode for current Midi Control Out module only - button

Turn on Solo Mute All to mute all Midi Control Out modules - button

Solo Mode text box - this will display one the following messages:

-"Midi Control Out Solo is currently Off (default setting)",

-"Midi Control Out Solo is currently On for this module only,so nothing will be output from other Midi Control Out modules",

-"Midi Control Out Solo is currently On for another module so nothing will be output from this module",

-"Solo Mute All is On so nothing will be output from any Midi Control Out modules"

Copy current Port to all Midi Control Out modules - button 

Copy current Data Rate to all Midi Control Out modules - button 

Turn Active on for all Midi Control Out modules - button 

Turn Active off for all Midi Control Out modules - button
