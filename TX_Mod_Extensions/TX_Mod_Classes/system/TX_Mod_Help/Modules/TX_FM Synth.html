FM Synth


Module type: Audio Source (mono output, triggered)


Introduction:

This module creates polyphonic synthesised notes using multi-layer Frequency Modulation (FM) Synthesis.

It uses 6 "operators" (which are the equivalent to oscillators), which can each modulate the frequency of any of the other 6 operators, including self-modulating. The output level of each operator can also be set (see controls below).

Notes can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:


Operators button

Main level - this sets the overall output level of the synth

Op 1 ... Op6/ All Ops buttons - these select which operator is being edited. The final one shows all 6 operators.

Freq ratio - this sets the frequency of the current operator in relation to the midi note played. For example, if the frequency ratio is 1, the operator's frequency will be the same as the midi note played. Whereas if the frequency ratio is 0.5, the operator's frequency will be an octave below the midi note played.

Phase - this sets the starting phase of the current operator.

Amplitude - the amplitude of the operator affects how much it can modulate the frequency of other operators. It also affects

Mod Op 1 - 6 - these set the amount of frequency modulation coming from operators 1- 6 to affect the frequency of the current operator.

Level - this sets the output level of the current operator


MIDI/ Note button

See help file MIDI Note Button for details.


Envelopes button

There are buttons to select one the following envelopes which are triggered when a note is played:

Vol Env - overall volume envelope for each note played

Op 1 Env - optional envelope which is added to the amplitude of Operator 1

Op 2 Env - optional envelope which is added to the amplitude of Operator 2

For Op 1 Env and Op 2 Env , the following controls are shown:

OFF/ON Op 1/2 Env - turns the envelope on/off.

Env amount - sets how much the envelope will affect the amplitude of Operator 1/2. The range is -1 to +1. Negative values will mean the envelope moves the amplitude in a negative direction.

For other envelope controls, see help file Env DADSR.
