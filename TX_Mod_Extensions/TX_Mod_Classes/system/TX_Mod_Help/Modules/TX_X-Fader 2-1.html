X-Fader 2-1


Module type:  Audio Source (mono output)


Introduction:

This module crossfades between two mono audio input signals. 


Controls:

Level 1 - sets the level of input 1. 

Level 2 - sets the level of input 2. 

X-Fade - the mix between inputs 1 & 2. 


Audio Input Busses:

Input 1  

Input 2  


Control Modulation Input Busses:
