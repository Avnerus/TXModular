Wave Terrain


Module type:  Audio Source (mono output)


Introduction:

This module outputs a synthesised waveform as a mono audio signal. 

The synthesis is based on a mathematical model of an object moving over a 3D terrain. The vertical movement created by orbiting around the terrain at an audio frequency creates an output waveform. 

The orbit can be varied (using sliders) which generates time varying waveforms. Different mathematical functions can be used to define the shape of the terrain and the orbit that is used to move around it. 

The edit pages for this module include text boxes which show the underlying computer code, written in SuperCollider, which define the Terrain and Orbit used for the synthesis.

If you are not familiar with SuperCollider code, you can ignore the computer code and use the various presets to select different Terrains and Orbits.


For further details and technical info about Wave Terrain Synthesis, see:

- http://www.propheticdesire.us/microsound/html/2002/2002-12/msg00152.html

- http://www.csounds.com/jmc/Articles/Wts/WTS.html

- The Computer Music Tutorial, by Curtis Roads, pp.163-167
