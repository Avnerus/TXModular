Control Delay


Module type:  Control Insert


Introduction:

This module delays a control signal. 


Controls:

Maximum time - this allocates more computer memory if you need longer delay times. Up to 5 minutes delay time can be set. 

Tap tempo button - this displays measures the tempo when it is clicked with a mouse on each beat. Use the Command button to drag the BPM to the Delay time BPM.

Delay time - this is shown in milliseconds as well as beats per minute (bpm) 

time x 2 , time x 3 , time / 2 , time / 3  - buttons to change the delay time. 

Dry-Wet Mix - this sets the mix of the original signal with the delayed signal.  This will usually be set to 1 - i.e. 100% delayed signal.


Control Modulation Input Busses:

Delay time  

Dry-Wet Mix
