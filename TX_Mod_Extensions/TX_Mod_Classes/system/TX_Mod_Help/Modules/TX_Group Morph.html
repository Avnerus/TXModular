Group Morph


Module type:  Control Source


Introduction:

This module outputs 15 control signals, all at values between 0 and 1. It was designed for controlling multiple parameters of other modules.

The parameters are set in 5 different groups (A/B/C/D/E). Then you can control the morphing between the different groups.

Note: Use a Meter (see top right corner of main window) if you need to display the output signals when using the group morph.


Controls:


Group Morph button

Output -  sets the outputs to use the Group Morph settings or else Solo one of the Groups A/B/C/D/E. In normal use, it should be set to Group Morph. The others choices are only used when changing the settings for a particular group.

Morph Type - select between different types of morph, 2-D or Linear, using 2 to 5 groups. 

Morph X-Y / Morph X - depending on the Morph Type chosen either a 2D slider or a normal slider will be displayed for morphing between different Groups.


Group A button

Output -  sets the output.

Randomise - this button sets A1-A15 to random values

Name - each of the outputs can be named for clarity

Value A1...A15 - sets the value for each output 1-15

Store Snapshot - sets which snapshot number is stored

Name - sets the snapshot name

Load Snapshot - sets which snapshot number is loaded


Group B/Group C/ Group D buttons - see Group A Button


Control Modulation Input Busses:

X Value 

Y Value
