Env Curve


Module type:  Control Source (triggered)


Introduction:

This module creates user-defined envelopes as control signals. The curve can be edited with a mouse.

They can be triggered in various ways such as by MIDI notes, by other modules (such as Step Sequencer,  Audio Trigger) or by action buttons on an interface screen. 


Controls:

Env Curve - this sets the shape of the curve.  The curve can be edited with a mouse.

Invert - turns the curve upside down. 

Reverse - reverses the curve from right to left. 

Smooth - evens out spikes or sudden changes in the curve. Useful when curve is drawn by the mouse.

Reset - resets the curve to a linear curve.

Load 1/2/3/4/5 - to load curves stored in 5 slots.

Store 1/2/3/4/5 - to store curves in 5 slots.

Env Time - time taken for the whole envelope. 

Trigger Envelope - this will trigger an envelope whenever it is pressed.

Grid rows - no of rows (1-99) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-99) separated by white lines in the grid behind the curve.

Midi Listen - this allows the module to be triggered by a midi note. 

Midi Chan. - set the midi channels the module will listen to. 

Note range - which notes will trigger the module.

Vel range - which velocities will trigger the module.

Scale level to velocity - if this is set on, then the maximum level of the envelope will depend on the velocity of the trigger or midi note; if this is set off,  the maximum level will always be100%.

Polyphony - maximum number of simultaneous envelopes.
