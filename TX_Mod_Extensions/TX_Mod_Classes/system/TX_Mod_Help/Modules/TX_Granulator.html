Granulator


Module type: Audio Source (stereo output, triggered)


Introduction:

This module takes a mono sample from the sample bank and plays small "grains" from it which are panned in stereo.

It can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:


Sample button

Mono sample - select from all the mono samples in the system's sample bank.

Reverse play - if on, the playback of the grains will be reversed.

Add samples to bank - to add any audio samples stored on the hard disk.

Show/Hide - buttons for the display of the sample waveform. When the waveform is displayed, you can zoom in and out using the zoom buttons. You can also select which part of the waveform is used for granulation by using the select all/ select none buttons.

Play range - which part of the sample will be used for grains.

Play position - the current position in the sample that grains are taken from

Vary position - this adds a random variation to the play position. If set to zero, all grains will be taken from the same play position.

Grain time ms - how long each grain is. Shorter times sound more fragmented.

Grain density - the number of overlapping grains. Values below one will have gaps between grains.

Vary pitch - this adds a random variation to the pitch of each grain. If set to zero, all grains will be at the same pitch.

Vary timing - this adds a random variation to the start time of each grain. If set to zero, all grains will be at equally spaced times.

Pan range - each pitch will be placed randomly across the pan range. Select presets using the black triangle at the end of the row.

Level - the output level of the granulator.


MIDI/ Note button

See help file MIDI Note Button for details.

Note: unlike most other audio synth or sample modules, keyboard tracking is turned off by default in the Granulator module. Of course, you can turn it on if required.


Envelope button

See help file Env DADSR for details.
