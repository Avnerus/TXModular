LFO Curve


Module type:  Control Source


Introduction:

Low Frequency Oscillator with an editable waveform curve - using a mouse. This module can be used for modulating parameters of other modules. 


Controls:

Frequency - the rate of change of the waveform

LFO Curve - the curve used for the waveform. You can edit the curve with a mouse.

Invert - turns the curve upside down. 

Reverse - reverses the curve from right to left. 

Smooth - evens out spikes or sudden changes in the curve . Useful when the curve is drawn by the mouse or when noise has been mixed in, since it reduces the harshness of the distortion. 

Reset - resets the curve to a linear curve.

Load 1/2/3/4/5 - to load curves stored in 5 slots.

Store 1/2/3/4/5 - to store curves in 5 slots.

Rebuild curve by mirroring - this button can be useful for creating symmetrical curves. The curve is mirrored at the right hand edge.

Rebuild curve by mirroring & inverting - for this button, the curve is mirrored at the right hand edge, inverted and finally the whole curve is normalised.

Grid rows - no of rows (1-99) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-99) separated by white lines in the grid behind the curve.

Output range - preset output ranges including negative values


Control Modulation Input Busses:

Frequency
