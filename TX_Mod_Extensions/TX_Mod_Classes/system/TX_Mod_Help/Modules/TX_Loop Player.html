Loop Player


Module type: Audio Source (mono output, triggered)


Introduction:

This module takes a mono loop from the loop bank and plays it back .

It can be triggered in various ways such as by MIDI notes, other modules (e.g. Step Sequencer, Action Sequencer, Audio Trigger) or action buttons on an interface screen.


Controls:


Loop button

Mono loop - select from all the mono loops in the system's loop bank. Stereo loops from the loop bank won't be shown.

Loopbank Settings - the settings for original bpm and total beats of the loop are displayed, taken from the settings in the loopbank. These can be edited in the loopbank screen.

Add loops to loop bank - to add any audio loops stored on the hard disk.

Show/Hide - buttons for the display of the loop waveform. When the waveform is displayed, you can zoom in and out using the zoom buttons. You can also select which part of the waveform is played by using the select all/ select none buttons.

Start/End - the start and end of the loop section that will be played. These are shown as fractions (e.g. 7/8), as well as numbers between 0 and 1.

Fractions can be useful for selecting particular sections of rhythmic loops - such as 2/12 to 8/12.

Loop type - there are two options:

looped - play looped from start point to end point - this is the default setting.

single shot - play once from start point to end point

Reverse - if on, loop playback will be reversed.

Level - the output level of the loop player.

Play BPM - The BPM that the loop will be played back at. The pitch of the loop will go up or down compared to the original loop, depending the how different the play bpm is from the original bpm. If the play bpm is lower, the pitch will also be lower.

Speed factor - A quick way to change the playback speed.

Higher numbers will increase the speed, lower numbers will decrease the speed.

A setting of 0 will be normal speed, A setting of 1 will be double the speed, A setting of -1 will be half the speed,


MIDI/ Test button

Midi Listen - this allows the module to be triggered by a midi note.

Midi Chan. - set the midi channels the module will listen to.

Note range - which notes will trigger the module.

Vel range - which velocities will trigger the module.

Test Loop Settings: - these set the velocity and play time of the test loop that is played when the Play test loop button is pressed.


Play test loop button

This starts the loop playing.

The velocity and time of the loop that is triggered are set using the MIDI/ Test button.


Stop test loop button

Stops the test loop playing.
