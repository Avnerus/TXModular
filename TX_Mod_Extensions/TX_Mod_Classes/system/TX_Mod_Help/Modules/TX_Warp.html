Warp


Module type:  Control Insert


Introduction:

This module is for "mapping" or changing the behaviour of a control signal.

The incoming signal is mapped using a transfer curve which is displayed on the screen.

The horizontal (left-right) axis of the curve represents the range of values of the input signal (based on the Input range control).

The vertical (up-down) axis of the curve represents the range of values of the output signal (based on the Output range control).

The default curve is a linear curve which will have no effect on the incoming signal.

There are 5 slots for storing curves - useful if you are trying out different curves.


Controls:

Input range - limits the range of value coming in. Values outside the range will be forced to be within range. 

Warp curve - sets a transfer curve which maps the input signal of the warp module to new output values.

Invert - turns the curve upside down. 

Reverse - reverses the curve from right to left. 

Smooth - evens out spikes or sudden changes in the curve. Useful when curve is drawn by the mouse.

Reset - resets the curve to a linear curve.

Load 1/2/3/4/5 - to load curves stored in 5 slots.

Store 1/2/3/4/5 - to store curves in 5 slots.

Preset curves - select a presets to load it.

Grid rows - no of rows (1-99) separated by white lines in the grid behind the curve.

Grid cols - no of columns (1-99) separated by white lines in the grid behind the curve.

Rebuild curve by mirroring - this button can be useful for creating symmetrical curves. The curve is mirrored at the right hand edge.

Rebuild curve by mirroring & inverting - for this button, the curve is mirrored at the right hand edge, inverted and finally the whole curve is normalised.

Output range - sets the range of values that the output signal will be mapped to.
