Waveshaper


Module type:  Audio Insert (mono input, mono output)


Introduction:

This module changes or distorts to the input signal. It changes the shape of the signal by using a function curve which can be drawn by hand or generated automatically based on harmonics and noise. 


Controls:


Settings page button

Shaping type - sets type of waveshaping. Can be variable, which changes with the amplitude of the signal, or constant . 

Curve - the curve used for waveshaping. You can edit the curve with a mouse. Some presets which compress or expand the signal are included.

Invert - turns the curve upside down. 

Smooth - evens out spikes or sudden changes in the curve . Useful when the curve is drawn by the mouse or when noise has been mixed in, it can reduce the harshness of distortion. 

Reset - resets the curve to a linear curve.

Load 1/2/3/4/5 - to load curves stored in 5 slots.

Store 1/2/3/4/5 - to store curves in 5 slots.

Rebuild curve by mirroring & inverting - this can be useful for creating waveshaping curves.  The curve is mirrored at the bottom left corner and then the left hand side is inverted. Finally the whole curve is normalised.

In Gain - cuts/boosts the signal before distorting it. This can affect how much distortion occurs. 

Out Gain - useful for reducing the volume of the distorted signal. This allows the dry and wet signals to be balanced in volume. 

Dry-Wet Mix - this sets the mix of the original signal with the affected signal.  For send effects this will usually be set to 1 - i.e. 100% affected signal.


Harmonics page button

Shaping type, Curve, Invert, Smooth, Reset,  ... same as above

Harmonics - sets the level for 16 harmonics that can be mixed into the curve.

Harm Mix - this sets the mix of the harmonics with the original curve.  

Add harmonics to the curve - this button mixes in the harmonics (based on the Harmonics and Harm Mix settings). 


Noise page button

Shaping type, Curve, Invert, Smooth, Reset, ... same as above

Noise Mix - this sets the mix of the noise with the original curve.  2 types of noise can be used. Tapered noise will affect louder signals more than quieter ones (similar to amplifier distortion). Untapered noise affects all signals in the same way.

Add tapered noise to the curve - this button mixes in tapered noise (based on the Noise Mix setting). 

Add untapered noise to the curve - this button mixes in untapered noise (based on the Noise Mix setting). 


Control Modulation Input Busses:

Dry-Wet Mix
