File Recorder St


Module type:  Audio Insert (stereo input, silent stereo output)


Introduction:

This module records stereo audio signals and writes them to aiff or wave files on the hard disk. 

The stereo outputs are silent so the module can be connected to any output bus.


Controls:

Record format - for the recorded file. 

Record Time - how long to record for displayed in seconds or beats at a particular tempo/BPM. If the recording time is set to 0, the recording will continue until the Stop rec button is pressed or the module is told to stop recording. 

Left rec level - can be used to fade in or out during recording. 

Right rec level - can be used to fade in or out during recording. 

Select file - button to name the file to be recorded to.

Start rec - button to start the recording.

Stop rec - button to stop the recording.

Sync Start - if this is turned on, the recording can be started by pressing the Sync Start button. This is useful if you want the recording to begin at the same time as sequencers are started.

Sync Stop - if this is turned on, the recording can be stopped by pressing the Sync Stop button. This is useful if you want the recording to end at the same time as sequencers are stopped.

Record status - gives information about the recording process. 

Record clock - how many seconds have been recorded so far. 

Current file - the name of the file currently set for recording. 

Previous file - the name of the previous file recorded.
