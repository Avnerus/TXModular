The TX Modular System - Index of Help Files


Learning to use the TX Modular System:

To learn how to use the TX Modular System, please read help files TX_1 to TX_5 (see section Getting Started).

These help files introduce how the TX Modular works and take you through building a system.

They also explain how to open example files containing previously built systems. A quick way to learn how to use the TX Modular is to explore and modify example systems.
